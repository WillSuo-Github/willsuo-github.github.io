Pre-Beta macOS Utility

Keep Downloads usable without building a Hazel project.

DropSort is a planned Mac app for people whose Downloads folder turns into low-grade daily cleanup. The first version is intentionally narrow: watch a folder, suggest starter rules, preview the move plan, and let you undo the last sort run.

What the first build is supposed to do

  • Watch Downloads or another selected working folder.
  • Offer a few practical starter rules for common clutter types.
  • Preview where files will move before the action runs.
  • Let you undo the latest sort run if the result is wrong.

What it is not pretending to do

  • No account requirement in the planned MVP.
  • No cloud sync promise.
  • No giant automation system on day one.
  • No claim that a public beta or App Store build exists yet.
Early list: Email to join the pre-beta list

Who this is for

Mac users who repeatedly clean up screenshots, PDFs, installers, ZIP files, exports, and stray documents.

Positioning

Simpler than Hazel for the single job of keeping Downloads and work-in-progress folders under control.

Pricing hypothesis

The current test is a one-time paid utility around $9.99, not a subscription.
